Сканер портов
Программа просит ввести хост который будет сканироватся
Далее запрашиваются два числа, начало и конец диапазона сканирования
Далее создается tcp сокет для каждого порта и устанавливается соединение по этому порту
Если не произошло ошибок при соединении значит порт открыт, что и пишется в консоль
